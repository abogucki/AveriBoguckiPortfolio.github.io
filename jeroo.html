Unit 2

In this Jeroo unit, I now knew how to use Jeroo actions to solve a problem. However, this unit also introduced me to Jeroo methods so I wouldn't have to repeat single actions over and over again. Precision was a key element for these methods. This unit is the unit where I started learning more complicated methods involving the Jeroos.

Concepts Learned

  • I learned how to use method calls to solve problems
  • I also learned that I must get comfortable with single Jeroo methods being called multiple times in solving problems that involve the same actions being repeated
  • These labs require precision in making sure the Jeroos accomplish their assigned job

Unit 3

In this Jeroo unit, I now knew how to use simple Jeroo methods to solve a problem. This unit was all about planning out a method to solve the problem with precision and accuracy. You must always meet the pre and post conditions. By the end of this unit I knew how to do more complicated Jeroo methods and solve more complicated problems.

Concepts Learned

  • I learned how to analyze a problem and determine how to break it into smaller tasks and then create the needed methods to solve it
  • I also learned that I must plan each method based on preconditions as well as post conditions
